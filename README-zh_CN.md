# Browser Scripts

[English](README.md) | 简体中文

这个仓库包含了多个浏览器脚本，主要包括油猴(Tampermonkey)脚本和浏览器控制台脚本，用于提升日常浏览体验和工作效率。

## 油猴脚本

油猴(Tampermonkey)是一款流行的浏览器扩展，允许用户添加和管理自定义脚本。要使用本仓库中的油猴脚本，请先安装油猴插件：

- Chrome 浏览器
  - [Chrome 网上应用店](https://chromewebstore.google.com/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- Firefox 浏览器
  - [Firefox 附加组件商店](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

### Autofill Wordpress

用于自动填写 WordPress 后台表单的实用脚本，提升内容管理效率。

详细功能与使用说明请查看 [使用文档](./tampermonkey/autofill-wordpress/README-zh_CN.md)。
